Wanderlust

Wanderlust ist ein kleines Projekt, das die Wanderwege als einzelne geoJSON und GPX Dateien zur Verfügung stellt. Es nutzt dafür die offenen Daten

Das Projekt ist auf GitHub verfügbar und freut sich über Pull Requests. Die Seite nutzt PHP.

Wanderwege in Wien

Die Stadt Wien bietet alle Wanderwege der MA49 als offene Daten in einer Datei an. Dieses Projekt teilt die Wanderwege in einzelne Dateien auf und stellt sie zuf Verfügung. Es ist geplant in einer zukünftigen Version Wanderwege wie den Rundumadum-Wanderweg zu einer Datei zusammenzufassen - als zusätzliches Angebot.

Datenquelle: Stadt Wien – https://data.wien.gv.at

Die Wanderwege können einzeln oder als TAR-Ball (gzip-komprimiert) herunter geladen werden: [GeoJSON] [GPX]
